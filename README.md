# Instalytics

A privacy-focused Instagram follower analyzer. All data is processed locally in your browser - nothing is uploaded to any server.

## Features

- Identify who doesn't follow you back
- Find your fans (followers you don't follow)
- See mutual followers
- Export results to CSV or JSON
- Search and filter username lists
- Sort by name or date
- Copy usernames to clipboard
- View follow statistics and charts
- Save snapshots to compare changes over time
- Support for multiple follower files (large accounts)
- Mobile-friendly design

## How to Use

1. Go to Instagram Settings > Account Center > Your Information and Permissions
2. Click "Download your information" > Request a download
3. Select JSON format (not HTML)
4. Wait for email (usually takes a few hours)
5. Download and extract the ZIP file
6. Upload `followers_1.json` and `following.json` to Instalytics

## Development

This is a single HTML file with no dependencies. Just open `index.html` in a browser.
